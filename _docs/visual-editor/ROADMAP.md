# Visual Editor — Жол Картасы (Roadmap)

## ✅ Фаза 1-3: Фундамент жана Текст (Completed)
- [x] `EditorContext` — Глобалдык абал жана тарых (Undo/Redo).
- [x] `Toolbar` — 4 табдуу профессионалдык панель.
- [x] Тексттик элементтерди толук башкаруу (Font, Style, Shadow, Align).
- [x] Grid Snap (Сеткага байлоо).
- [x] Layers тизмеси жана Visibility (көз) башкаруу.

## ✅ Фаза 4-5: Интерактивдүүлүк жана Физика (Completed)
- [x] **3-State Interaction:** None, Modal, Portal Link.
- [x] **Animation Motor:** Speed, Amplitude, Noise башкаруу.
- [x] **Sticky System:** Элементтерди бири-бирине байлоо (`parentId`).
- [x] **Organic Lag:** Бала-элементтердин атасын ээрчүү физикасы (`spring` + `stickyLag`).
- [x] **Duplicate Tool:** Элементтерди бардык параметрлери менен көчүрүү (📋).

## ✅ Фаза 6-9: Активдер, Группалар жана Скролл (Completed)
- [x] **Asset Management:** Сүрөттөрдү алмаштыруу, масштабдоо жана SVG түстөрүн башкаруу.
- [x] **Layers & Grouping:** Катмарларды иреттөө, группаларга бириктирүү жана папкаларды башкаруу.
- [x] **Background Editor:** Башкы фонду алмаштыруу жана анын атмосферасын (жарыктык, тунуктук) жөндөө.
- [x] **Long Page (Vertical Scroll):** Экрандан узун (Long Ribbon) карталарды жана вертикалдык скролду колдоо.

## ✅ Фаза 10: Multi-Scene Architecture (Deep Portals) (Completed)
- [x] **Dynamic Scenes:** Сценаларды JSON форматта түзүү, сактоо жана жүктөө системасы.
- [x] **Scene Hub:** Редактордун ичинде сценалар арасында заматта которулуу мүмкүнчүлүгү.
- [x] **Scene Binding:** "MODAL" элементтерин конкреттүү сценага байлоо (Порталдар).
- [x] **Cyrillic Support:** Сценалардын аттарына кыргыз тамгаларын жана боштуктарды колдонуу.
- [x] **Asset Sync:** Сүрөттү алмаштырганда анын координаталарын автоматтык түрдө нөлдөө (Predictable Reset).

## 🚀 КЕЛЕЧЕК ПЛАНДАР (Future Work)

> [!NOTE]
> **Техникалык концепция (Long Page Map):**
> Биздин система элементтерди **процент (%)** менен сактагандыктан, картанын бийиктиги (Height) канчалык өзгөрбөсүн (мисалы, Фотошопто 5000px чейин узартылса), элементтер өз ордун жоготпойт. Фаза 9 ишке ашканда, карта контейнери `overflow: auto` режимине өтүп, колдоочуга "узун лента" сыяктуу картаны скролл кылып көрүүгө мүмкүндүк берет. Бул Фотошоптогу "Artboard" логикасына окшош иштейт.
