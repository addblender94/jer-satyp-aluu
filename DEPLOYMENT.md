# 🚀 Программаны жаңыртуу (Deployment Guide)

Эгер сиз программага (сайттын ичине) өзгөртүү киргизсеңиз, аны Интернеттеги (`.vercel.app`) сайтында жаңыртуу үчүн төмөнкү 3 кадамды аткарыңыз:

### 1. Локалдык текшерүү
Биринчи компьютериңизден баарын текшерип алыңыз:
```bash
npm run dev
```
Браузерден (localhost:3000) баары туура иштеп жатканын көргөндөн кийин, терминалды токтотуңуз (`Ctrl + C`).

### 2. GitHub'ка жүктөө (Push)
Терминалга ушул 3 команданы ирети менен жазыңыз:

```bash
# 1. Бардык өзгөртүүлөрдү белгилөө
git add .

# 2. Коммит жасоо (эмнени өзгөрткөнүңүздү кыскача жабыңыз)
git commit -m "Сценаны жаңыртуу же шрифттерди оңдоо"

# 3. Интернетке (GitHub) жөнөтүү
git push
```

### 3. Автоматтык жаңыртуу
Сиз `git push` кылгандан кийин, **Vercel** автоматтык түрдө сиздин жаңы кодуңузду алат жана 1-2 мүнөттүн ичинде сайтыңызды жаңыртат. 

---

### 💡 Маанилүү кеңештер:
- **Editor Mode:** Эгер сиз сайтты Интернеттен гана көрүүнү кааласаңыз, редакторду өчүрүп койсоңуз болот (Toolbar.tsx ичиндеги `isEditorMode` параметри).
- **Images:** Жаңы сүрөттөрдү кошсоңуз, аларды дагы GitHub'ка `git add .` аркылуу кошууну унутпаңыз.
- **Production Build:** Эгер ката болуп жатса, `npm run build` кылып каталарды алдын ала текшерсеңиз болот.
